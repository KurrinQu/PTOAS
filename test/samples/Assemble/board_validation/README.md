# Assemble NPU Validation (A3)

This test validates `pto.tassemble` functional correctness on board.

Semantics under test:

`dst[i + 8, j + 8] = src[i, j]` for `0 <= i,j < 16`.

Inputs/outputs:

- `v1.bin`: source tile (`16x16`, `float32`)
- `v2.bin`: destination tile (`32x32`, `float32`)
- `v3.bin`: output tile (`32x32`, `float32`)
- `golden_v3.bin`: expected output generated by CPU golden

Run on A3:

```bash
cd test/samples/Assemble/npu_validation
SOC_VERSION=Ascend910 ./run.sh
```

A passing run prints:

`[INFO] compare passed`
